# 学习页开发文档

## 页面目标

学习页负责把调音、节奏型、和弦转换、歌曲片段跟练串成一条新手学习路径。它不是单纯的课程列表，而是告诉用户“现在到哪一步、下一步做什么、完成后进入哪个练习”。

## MVP 范围

- 顶部保留互动课程定位，说明从零到第一首弹唱的路径。
- 展示必修课总体进度、当前课程、已完成课程数量。
- 展示下一步学习卡，卡片可直接进入课程详情。
- 展示必修路径节点：认识琴、调音、节奏、换和弦、歌曲片段。
- 展示学习主题快捷入口：调音基础、节奏型、和弦转换、歌曲片段。
- 保留必修 / 选修 / Pro 分类课程列表。

## 交互规则

- 点击当前课程或路径节点，打开课程详情弹层。
- 点击主题快捷入口，进入对应工具或页面。
- 课程完成后，学习页进度、当前课程、路径节点状态同步刷新。
- Pro 内容显示锁定状态，不阻断免费必修路径。

## 验证清单

- 打开预览页后，学习页能看到进度总览、下一步、路径节点、主题入口、课程列表。
- 必修路径当前节点和课程列表当前节点一致。
- 点击路径节点能打开课程详情。
- 点击“节奏型 / 和弦转换 / 歌曲片段”主题入口能进入对应练习或页面。
- Expo App 学习页消费 `practiceContent.courses` 和 `buildMvpCourseProgressPath`，而不是写死静态知识卡。
- Expo App 学习页课程主按钮复用根组件 `openCourse`：调音课进入调音器，练习课进入对应 PracticeRunner，歌曲课进入曲谱页。
- `npm.cmd test`、`npx.cmd tsc -p apps/mobile --noEmit`、preview script syntax、`git diff --check` 通过。
