# 我的页 / 记录页开发文档

## 页面目标

我的页负责承接用户练习后的复盘体验，让新手看到自己练了多久、完成了哪些课程、最近分数如何、下一步该练什么。它是调音、节奏型、和弦转换、歌曲跟弹等模块的结果汇总页。

## MVP 范围

- 顶部展示用户身份、等级、最近得分和经验进度。
- 展示连续天数、今日练习分钟、已掌握和弦。
- 展示必修学习路径进度和当前下一课。
- 展示最近成绩摘要：最近得分、最高分、练习次数。
- 展示下一步建议，入口指向学习页或练习页。
- 展示打卡日历、成就墙、最近练习记录。
- 支持清空本地最近练习记录，便于预览测试。

## 交互规则

- 练习结束后，最近练习、今日分钟、最近得分同步刷新。
- 学习进度根据 `courseProgress` 自动计算。
- 点击下一步建议进入学习页。
- 点击清空记录只清空本地 `sessions`，不重置课程进度。

## 验证清单

- 我的页能看到统计、学习进度、成绩摘要、建议、日历、成就、最近记录。
- 完成练习后，最近记录和分数会更新。
- 清空最近记录后，最近记录区域显示空状态。
- Expo App 我的页消费 `practiceHistorySummary`、`practicePathSummary`、`buildCoursePath` 和 `nextPracticeRecommendation`，与 preview 记录页保持同一信息结构。
- Expo App 下一步建议入口回到学习页，不直接绕过课程路径。
- `npm.cmd test`、`npx.cmd tsc -p apps/mobile --noEmit`、preview script syntax、`git diff --check` 通过。
