# AI 尤克里里弹唱 SDD

版本：v0.2
日期：2026-07-12
文档类型：Screen / System Design Document

## 1. 设计目标

本阶段设计目标是把产品从“功能堆叠”调整为“新手能持续练下去的弹唱练习 App”。

设计关键词：

- 手机 App 感。
- 一屏优先。
- 指法图优先。
- 节拍驱动。
- 练习闭环。
- 清晰、轻量、可继续扩展。

页面不追求营销式大图，也不做长篇说明。用户进入页面后应立刻知道当前能做什么、下一步该点哪里、现在该弹什么。

## 2. 信息架构

底部固定五 Tab：

```text
首页 | 练琴 | 曲谱 | 学习 | 我的
```

页面关系：

```text
首页
  ├─ 调音器快捷入口
  ├─ 和弦库快捷入口
  ├─ 曲谱库快捷入口
  └─ 练习快捷入口

练琴
  ├─ 智能调音器
  ├─ 节拍器
  ├─ 和弦库
  ├─ 节奏练习
  └─ 和弦转换

曲谱
  ├─ 歌曲列表
  ├─ 歌曲详情
  ├─ 单音跟弹
  └─ 和弦跟弹

学习
  ├─ 基础知识
  ├─ 图文教程
  └─ 相关练习跳转

我的
  ├─ 打卡
  ├─ 练习统计
  ├─ 收藏
  ├─ 历史记录
  └─ 设置
```

## 3. 视觉方向

整体风格：

- 明亮、干净、偏 App 工具体验。
- 使用柔和背景和高对比主操作按钮。
- 音乐练习页面可以有轻微拟物感，但不堆装饰。
- 卡片圆角适中，信息密度比旧版更高。

建议色彩：

- 主色：清爽蓝或青绿色，用于当前状态、选中态、主要入口。
- 强调色：橙色或黄色，用于开始练习、播放、重音。
- 警示色：红色，用于偏差、错误、危险操作。
- 背景：浅灰蓝或暖白，避免纯白刺眼。

限制：

- 不使用大面积单一深色背景。
- 不用长页面堆卡片。
- 不只用英文和弦代码表达练习内容。

## 4. 通用组件

### 4.1 底部 Tab

要求：

- 固定在底部。
- 五个入口文字清晰。
- 当前 Tab 高亮。
- 点击区域适合手指。

### 4.2 和弦指法图

所有和弦场景复用同一视觉规则：

- 顶部显示和弦名。
- 四根弦按 G C E A 标注。
- 品格网格清晰。
- 空弦、按弦、不弹状态明确。
- 圆点内显示手指编号。

尺寸：

- 大图：用于当前和弦。
- 中图：用于和弦库卡片。
- 小图：用于预告、歌曲列表、练习序列。

### 4.3 节拍点

要求：

- 第一拍为重音点，使用红色或更强颜色。
- 轻拍使用蓝色、灰蓝或次级颜色。
- 当前拍有放大、描边或亮度变化。
- 可支持 3/4、4/4、6/8。

### 4.4 模块入口卡

用于首页和练琴页：

- 图标 + 标题 + 简短副标题。
- 不放长说明。
- 状态可选：推荐、最近、已完成。

### 4.5 练习结果卡

用于练习结束：

- 总分。
- 节奏命中。
- 提前/滞后趋势。
- 下一步建议。
- 重新练习和下一项按钮。

## 5. 页面设计规格

### 5.1 首页

首屏结构：

1. 顶部：App 名称 `AI 尤克里里弹唱`，右侧可显示连续天数或头像。
2. 打卡区域：紧凑 contribution grid。
3. 四个模块入口：调音器、和弦、曲谱库、练习。
4. 热门弹唱或今日推荐。
5. 底部 Tab。

设计要求：

- 375px 宽度下不需要读很多文字。
- 打卡区域必须紧凑，不抢主练习入口空间。
- 热门弹唱展示 2-3 条即可，更多进入曲谱页。

### 5.2 练琴页

结构：

1. 顶部标题：练琴。
2. 工具区：智能调音器、节拍器、和弦库。
3. 专项练习区：节奏练习、和弦转换。
4. 最近练习或推荐练习。

设计要求：

- 它是入口页，不展示完整练习报告。
- 每个入口点击进入独立页面。
- 工具和专项练习分组明确。

### 5.3 调音器页

结构：

1. 顶部：智能调音器 + 当前模式/标准调弦。
2. 三段状态：权限、电平、PitchFrame。
3. 中间：圆环或仪表式调音核心区。
4. 输入电平和噪声门限。
5. G/C/E/A 四弦状态。
6. 主按钮：开始/停止调音。
7. 次按钮：校准环境噪声。

调音核心区：

- 当前目标音名要足够大。
- cents 偏差显示明确。
- 已准状态用绿色。
- 偏高提示松弦，偏低提示拧紧。
- 稳定通过后自动跳到下一弦。

### 5.4 节拍器页

结构：

1. 顶部：节拍器。
2. BPM 大数字。
3. BPM 加减或滑杆。
4. 拍号/节奏模式选择。
5. 节拍点可视化。
6. 声音开关。
7. 开始/暂停按钮。

设计要求：

- 第一拍重音必须看得出来。
- 声音状态清晰。
- 控件不要过多，优先练习可用。

### 5.5 和弦库页

参考方向：用户提供的“和弦大全”样式。

结构：

1. 顶部标题：和弦大全。
2. 搜索框：搜索和弦，如 C、G7。
3. 分类 Tab：所有、大三和弦、小三和弦、属七和弦、挂留和弦。
4. 和弦网格：每张卡片含指法图、和弦名、试听按钮。
5. 收藏的和弦区域。

设计要求：

- 网格优先展示常用和弦。
- 卡片中指法图比文字更重要。
- 试听按钮不能干扰主要视觉。

### 5.6 节奏练习页

一屏结构：

1. 顶部：节奏练习 + 当前节奏型。
2. 节奏型选择横滑或紧凑列表。
3. BPM/拍号。
4. 中间：节拍点 + 当前扫弦动作。
5. 底部：开始/暂停、循环、声音开关。
6. 结果区域：练完后出现，不常驻占空间。

设计要求：

- 页面重点是“这一拍怎么弹”。
- 不把历史记录堆在当前练习页里。
- 评分结果可用弹层、折叠卡或完成页。

### 5.7 和弦转换页

一屏结构：

1. 顶部：和弦转换 + 当前练习组。
2. 和弦序列：小图展示待练和弦。
3. 中间左：当前和弦大指法图。
4. 中间右：下一和弦小指法图 + 倒计拍。
5. 节拍点。
6. 操作：开始/暂停、循环、BPM、声音。

设计要求：

- 当前和弦永远使用大指法图。
- 不只显示 `C Am F G7` 字符串。
- 切换逻辑跟随节拍。

### 5.8 曲谱页

结构：

1. 顶部：曲谱。
2. 搜索/筛选。
3. 歌曲列表。
4. 每个歌曲卡片显示：标题、难度、BPM、和弦、预计用时。

设计要求：

- 歌曲卡不展示受版权保护歌词。
- 和弦以小指法图或紧凑标签辅助展示。
- 点击进入详情。

### 5.9 歌曲详情页

结构：

1. 歌曲基本信息。
2. 使用和弦列表，展示小指法图。
3. 练习方式选择：单音跟弹、和弦跟弹。
4. 推荐先练内容。

### 5.10 单音跟弹页

结构：

1. 顶部：歌曲名 + BPM。
2. 时间轴/流水区：当前音居中，前后音预览。
3. 当前音：显示弦名和品格。
4. 节拍点。
5. 开始/暂停/循环。

设计要求：

- 新手能直接知道“哪根弦、几品”。
- 不只显示简谱或音名。

### 5.11 和弦跟弹页

结构：

1. 顶部：歌曲名 + 当前小节。
2. 当前和弦大指法图。
3. 后续和弦小卡片流水预告。
4. 节拍点。
5. 开始/暂停/循环。

设计要求：

- 当前和弦图必须足够大。
- 自动切换时有明确过渡。

### 5.12 学习页

结构：

1. 分类：入门、调音、节奏、和弦、弹唱。
2. 图文卡片。
3. 内容详情。
4. 相关练习入口。

设计要求：

- 图文服务实际练习。
- 每篇内容都能引导到调音、节奏或和弦练习。

### 5.13 我的页

结构：

1. 用户/本地身份区域。
2. 连续打卡。
3. 练习统计。
4. 最近记录。
5. 收藏和弦/歌曲。
6. 设置。

设计要求：

- 数据口径与首页一致。
- 不登录也能记录本地练习。

## 6. 交互状态

通用状态：

- 默认。
- 当前。
- 已完成。
- 锁定。
- 推荐。
- 错误/偏差。
- 加载中。

调音状态：

- 未授权。
- 预览模拟。
- 读取电平。
- 等待拨弦。
- 检测中。
- 已准。
- 偏高。
- 偏低。

练习状态：

- 未开始。
- 倒计时。
- 进行中。
- 暂停。
- 完成。
- 查看结果。

## 7. 响应式约束

目标设备：

- iPhone/Android 常见宽度：375px、390px、430px。
- Preview 优先模拟手机容器。

约束：

- 底部 Tab 不遮挡主按钮。
- 练习页核心操作不依赖长滚动。
- 和弦图在小屏不能被压缩到不可读。
- 文案超出时换行或缩短，不允许遮挡。

## 8. 可访问性

- 主要按钮高度不低于 44px。
- 文本对比度足够。
- 节拍不能只靠颜色区分，当前拍可使用大小、描边或动画。
- 声音开关状态必须有文字标识。
- 麦克风权限说明必须清晰。

## 9. 与现有实现的衔接

保留：

- `audio-core` 调音、pitch、scoring。
- 浏览器麦克风 PoC。
- 当前指法图绘制逻辑。
- 本地练习记录思路。

重做：

- 五 Tab 信息架构。
- 首页。
- 练琴入口页。
- 和弦库页面。
- 节奏练习页面。
- 和弦转换页面。
- 曲谱与歌曲跟弹页面。

迁移方式：

1. 先在 preview 中按新结构做完整体验。
2. 再把数据模型抽到 shared。
3. 最后让 Expo App 消费同样结构。

## 10. 实施顺序

第一批：页面骨架。

- 五 Tab。
- 首页。
- 练琴入口页。
- 曲谱、学习、我的占位但可点击。

第二批：核心工具。

- 调音器新样式。
- 节拍器独立页。
- 和弦库新样式。

第三批：专项练习。

- 节奏练习。
- 和弦转换。
- 练习结果。

第四批：歌曲练习。

- 曲谱库。
- 歌曲详情。
- 单音跟弹。
- 和弦跟弹。

第五批：App 对齐与真实音频。

- App 页面同步。
- 真实麦克风事件接入练习评分。
- 和弦真实识别评估。

## 11. 待确认问题

- 首页热门弹唱首批展示哪些歌曲或练习型片段。
- 和弦试听先用合成音还是先准备采样。
- 曲谱页是否需要收藏和最近打开。
- 我的页是否只做本地身份，还是预留账号入口。
- 学习页首批文章数量与主题。
