# AI 尤克里里弹唱 PRD

版本：v0.2
日期：2026-07-12
文档类型：Product Requirements Document
项目定位：以“AI 音乐学园”的公开互动学习形态为参考，做一款面向尤克里里新手的 AI 弹唱练习 App。

> 合规边界：本产品只借鉴公开可见的学习路径、工具结构和交互模式，不复制竞品品牌、课程、曲谱、视频、图片、音频、文案和会员内容。曲谱与歌曲练习优先使用自有、无版权或已授权内容。

## 1. 方向重置

当前产品不再以“调音器 + 节拍器 + 四和弦跟练”的零散工具集合为主线，而是转为“实际弹唱练习”主线。

新的产品主线：

1. 首页告诉用户今天该练什么。
2. 练琴页提供调音、节拍器、和弦库、节奏练习、和弦转换等练习入口。
3. 曲谱页承载歌曲练习，支持单音跟弹与和弦跟弹。
4. 学习页承载基础知识，用图文方式解释乐理、指法、节奏和弹唱技巧。
5. 我的页承载打卡、练习数据、记录、收藏和设置。

已有的音频算法、调音判定、PitchFrame、练习记录能力继续保留，但后续页面与功能要围绕新信息架构重做。

## 2. 目标用户

核心用户：

- 零基础或初级尤克里里学习者。
- 想从调音、节奏、和弦转换过渡到真实歌曲弹唱的用户。
- 希望用手机完成练习、反馈、记录和复盘的自学用户。

典型问题：

- 不知道每天先练什么。
- 看见 C、G、Am、F 这样的英文代码，不知道如何按。
- 节奏感弱，需要节拍辅助。
- 换和弦慢，不能跟上歌曲。
- 会几个和弦，但无法连接成歌曲弹唱。

## 3. 产品目标

P0 目标：

- 形成新的五 Tab 信息架构：首页、练琴、曲谱、学习、我的。
- 完成移动端一屏优先的主页面设计。
- 练琴页拆分为独立工具与专项练习入口。
- 所有和弦相关页面都展示指法图，不只展示英文代码。
- 调音器继续可用，并按新页面样式重做。
- 节拍器有声音、有重音、有不同节拍模式。
- 节奏练习和和弦转换练习可跟随节拍循环，并有练习评分。

P1 目标：

- 曲谱库支持歌曲详情。
- 歌曲详情支持单音跟弹与和弦跟弹两种练习模式。
- 跟弹页面使用类似流水/时间轴的变化方式展示下一步要弹的内容。
- 我的页展示练习记录、连续打卡、收藏和历史成绩。

P2 目标：

- 接入真实麦克风评分，提升节奏、音准、扫弦触发、和弦识别可信度。
- 增加更多练习内容、歌曲片段和个性化推荐。
- 支持本地内容扩展，后续可接内容后台。

## 4. 信息架构

底部五个 Tab：

| Tab | 主要职责 |
| --- | --- |
| 首页 | 今日目标、连续打卡、四个快捷模块、热门弹唱、最近练习 |
| 练琴 | 调音器、节拍器、和弦库、节奏练习、和弦转换 |
| 曲谱 | 歌曲列表、歌曲详情、单音跟弹、和弦跟弹 |
| 学习 | 基础知识、图文教程、弹唱技巧 |
| 我的 | 打卡、练习统计、收藏、历史记录、设置 |

## 5. 页面需求

### 5.1 首页

页面名称：AI 尤克里里弹唱。

首页必须包含：

- 连续打卡展示，可使用类似 GitHub contribution grid 的紧凑日历。
- 四个主要模块入口：调音器、和弦、曲谱库、练习。
- 热门弹唱列表，展示歌曲名、难度、推荐练习方式、预计用时。
- 最近练习或今日建议。
- 内容尽量在首屏内清晰展示，避免过长滚动。

验收标准：

- 用户打开 App 后 3 秒内能知道下一步该练什么。
- 首页入口能跳到对应功能页。
- 移动端 375px 宽度下核心内容不遮挡、不溢出。

### 5.2 练琴页

练琴页是工具与专项练习的入口页。

必须包含：

- 智能调音器。
- 节拍器。
- 和弦库。
- 节奏练习。
- 和弦转换。

每个入口点击后进入独立页面，不在练琴页堆叠所有功能。

验收标准：

- 练琴页像功能总览，而不是长练习报告页。
- 用户能明确区分“工具”和“练习”。

### 5.3 调音器

调音目标：尤克里里标准 GCEA。

必须包含：

- 新样式调音界面。
- 当前目标弦/音名。
- 圆环或仪表式音准反馈。
- 权限、电平、PitchFrame 三段状态。
- 输入电平条和环境噪声校准。
- 自动屏蔽环境噪声，尽量只响应拨弦。
- 每根弦稳定调准后自动跳到下一根弦，并可循环。

判定规则沿用当前稳定版本：

- ±8 cents 内：已准，不用调。
- 8-12 cents：接近。
- 超过 12 cents：提示拧紧或松弦。

验收标准：

- 静态预览页可模拟完整流程。
- 浏览器麦克风 PoC 保留真实拨弦检测入口。
- Expo App 后续接真实 AudioEngine，不影响页面结构。

### 5.4 节拍器

必须包含：

- BPM 调节。
- 拍号/节拍模式选择。
- 第一拍重音，后续轻音。
- 声音开关。
- 可视化节拍点。
- 开始、暂停、重置。

验收标准：

- 点击开始后能听到节拍声。
- 第一拍和轻拍音色或音高明显不同。
- 页面可用于辅助节奏练习和和弦转换练习。

### 5.5 和弦库

必须包含：

- 和弦搜索。
- 分类筛选：全部、大三和弦、小三和弦、属七和弦、挂留和弦；也可按难度分级。
- 网格卡片展示和弦指法图。
- 收藏的和弦区域。
- 每个和弦有试听按钮。

声音要求：

- MVP 可先使用合成多音同时发声。
- 后续优先替换为真实尤克里里和弦采样。
- 不接受只播放单音来冒充和弦试听。

验收标准：

- 新手不看英文代码，也能通过指法图知道怎么按。
- C、G、Am、F、G7 等常见和弦优先可见。

### 5.6 节奏练习

节奏练习不再做成长页面，改为一屏优先。

必须包含：

- 节奏型选择。
- BPM 与拍号。
- 重音/轻音可视化。
- 当前要弹的扫弦方向：下扫、上扫、停顿、切音。
- 循环练习。
- 练习评分。

首批节奏型：

- 四拍下扫。
- 下 下 上 上 下 上。
- 下 下上 上下上。
- 分解/慢速入门节奏。
- 切音入门节奏。

验收标准：

- 用户能按节拍看懂何时扫弦。
- 练完能看到节奏分、命中次数、提前/滞后倾向。

### 5.7 和弦转换

和弦转换页面同样一屏优先。

必须包含：

- 顶部展示本次要练的和弦序列。
- 当前和弦必须展示指法图。
- 下一和弦展示小指法图或预告卡。
- 跟随节拍自动切换。
- 支持循环练习。
- 支持更多转换对，不只三组。
- 练习评分。

首批转换：

- C -> Am。
- Am -> F。
- F -> G7。
- C -> G。
- G -> Am。
- C -> Am -> F -> G7。

验收标准：

- 用户始终能看到当前要按的指法图。
- 切换时机由节拍驱动，而不是只靠手动点击。

### 5.8 曲谱库

必须包含：

- 歌曲列表。
- 难度、调式、BPM、使用和弦、预计练习时间。
- 歌曲详情。
- 进入单音跟弹或和弦跟弹。

内容策略：

- 优先使用无版权、自有改编、练习型旋律或已授权内容。
- 对熟悉流行风格的内容，只做原创练习片段，不复制受版权保护旋律和歌词。

验收标准：

- 用户点击歌曲后可以选择练旋律或练和弦。
- 歌曲练习与节奏、和弦库、练习记录打通。

### 5.9 单音跟弹

必须包含：

- 按节拍推进。
- 展示当前要弹的弦与品格。
- 展示接下来几个音。
- 类似流水/时间轴的自动变化。
- 支持多次自动循环。

验收标准：

- 新手知道“现在弹哪根弦、几品”。
- 页面不只显示音名，也显示可操作的弦/品格。

### 5.10 和弦跟弹

必须包含：

- 按节拍推进。
- 当前和弦展示完整指法图。
- 接下来和弦以小卡片预告。
- 类似流水/时间轴的自动变化。
- 支持循环练习。

验收标准：

- 用户在歌曲练习中仍然能看到指法图。
- 歌曲和弦练习可复用和弦转换练习的节拍与评分能力。

### 5.11 学习页

必须包含：

- 图文基础知识内容。
- 尤克里里结构、持琴、右手扫弦、左手按弦、调弦、节拍、和弦图读法等主题。
- 每篇内容可跳转到相关练习。

验收标准：

- 学习内容服务练习，不做孤立文章列表。

### 5.12 我的页

必须包含：

- 连续打卡。
- 累计练习时长。
- 最近练习记录。
- 收藏和弦/歌曲。
- 设置入口。

验收标准：

- 用户能看到自己的练习轨迹。
- 首页与我的页的数据口径一致。

## 6. 评分与记录

MVP 评分先以可解释为主：

- 节奏分：扫弦触发与节拍点偏移。
- 稳定分：连续命中、漏拍、提前/滞后趋势。
- 完成度：是否完成目标小节、循环次数、目标和弦或节奏型。
- 调音状态：每根弦是否达到 ±8 cents 内。

真实音频评分逐步接入：

1. 先接节奏 onset。
2. 再接单音 pitch。
3. 再接和弦 chroma/CQT 或模型识别。

## 7. 非功能要求

- 移动端优先，375px 和 430px 宽度必须可用。
- 练习页尽量一屏完成主要操作。
- 所有可点击区域适合手指点击。
- 和弦相关信息优先用图形表达。
- 静态预览页保持自包含，方便快速查看。
- 默认不上传用户原始录音。
- 需要麦克风时必须明确说明用途。

## 8. 里程碑

M1：文档与新信息架构确认。

- 更新 PRD/TDD/SDD。
- 明确五 Tab、页面清单、练习主线。

M2：静态预览新骨架。

- 首页、练琴、曲谱、学习、我的按新结构重做。
- 和弦库按新样式重做。
- 调音器按新样式接回已有调音能力。

M3：核心练习页。

- 节拍器。
- 节奏练习。
- 和弦转换。
- 练习评分与记录。

M4：歌曲练习。

- 曲谱库。
- 歌曲详情。
- 单音跟弹。
- 和弦跟弹。

M5：真实音频增强。

- 练习页接麦克风 onset。
- 单音 pitch 评分。
- 和弦识别评估。

## 9. 暂不做

- 完整会员系统。
- 云同步。
- 社区。
- 版权歌曲库。
- AI 视频课。
- 复杂多乐器扩展。

这些能力保留架构位置，但不进入当前 MVP。
